üìö Summary

In this section, we study vector-valued functions, extending the concept of real-valued functions to those that output vectors. This allows us to describe movements and changes in space, providing a crucial framework for analyzing motion in physics and engineering. Key topics include evaluating and graphing these functions, understanding their algebraic properties, and exploring displacement and average rate of change.

üéØ Main Results

  • Definition of Vector-Valued Functions: A vector-valued function is expressed as $\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$, where $f$, $g$, and $h$ are real-valued functions.
  • Evaluating and Graphing: Evaluating a vector-valued function at a specific $t$ involves evaluating each component function. The graph is the set of all terminal points of $\mathbf{r}(t)$ with initial points at the origin.
  • Algebra of Vector-Valued Functions: Operations such as addition, subtraction, and scalar multiplication are performed component-wise.
  • Displacement: The displacement vector $\mathbf{d}$ from $t = a$ to $t = b$ is given by $\mathbf{r}(b) - \mathbf{r}(a)$.
  • Average Rate of Change: For $\mathbf{r}(t)$, the average rate of change on $[a, b]$ is $\frac{\mathbf{r}(b) - \mathbf{r}(a)}{b - a}$.

üéØ Key Exercise Explanations

üìù Parameterizing a Curve

Problem: Parameterize the curve $y = x^2$ as a vector-valued function

Solution Method:

  1. Choose parameter: Let $t$ be our parameter (think of it as time)
  2. Express x in terms of t: $x = t$
  3. Express y in terms of t: Since $y = x^2$, we have $y = t^2$
  4. Write vector function: $\mathbf{r}(t) = \langle t, t^2 \rangle$

Answer: $\mathbf{r}(t) = \langle t, t^2 \rangle$ traces the parabola $y = x^2$

Note: Multiple parameterizations exist! For example, $\mathbf{r}(s) = \langle s, s^2 \rangle$ where $s = 2t$

üåÄ Cycloid Vector Function

Problem: Find a vector-valued function that models a cycloid (curve traced by a point on a rolling circle)

Solution Method:

  1. Start simple: Model point spinning around unit circle: $\langle \cos(\frac{3\pi}{2} - t), \sin(\frac{3\pi}{2} - t) \rangle$
  2. Clockwise rotation: Use negative $t$ to get clockwise motion
  3. Translate up: Add 1 to y-component to move circle above x-axis
  4. Rolling motion: Add $t$ to x-component to translate circle horizontally

Answer: $\mathbf{r}(t) = \langle t - \sin(t), 1 - \cos(t) \rangle$

Insight: This creates the famous cycloid curve seen in physics and engineering!

üìä Displacement Vector

Problem: Given $\mathbf{r}(t) = \langle \cos(t), \sin(t), t \rangle$ (3D helix), find displacement from $t = 0$ to $t = \pi$

Solution Method:

  1. Find initial position: $\mathbf{r}(0) = \langle \cos(0), \sin(0), 0 \rangle = \langle 1, 0, 0 \rangle$
  2. Find final position: $\mathbf{r}(\pi) = \langle \cos(\pi), \sin(\pi), \pi \rangle = \langle -1, 0, \pi \rangle$
  3. Calculate displacement: $\mathbf{d} = \mathbf{r}(\pi) - \mathbf{r}(0)$
  4. Compute: $\mathbf{d} = \langle -1, 0, \pi \rangle - \langle 1, 0, 0 \rangle = \langle -2, 0, \pi \rangle$

Answer: Displacement vector is $\langle -2, 0, \pi \rangle$

Physical meaning: Net change in position (not total distance traveled!)
