üìö Summary

In this section, we delve into the calculus of vector-valued functions. We extend the concepts of limits, derivatives, and integrals from real-valued functions to vector-valued functions, enabling us to analyze these functions with the tools of calculus. This section lays the groundwork for understanding how calculus applies to vector functions and sets the stage for exploring motion in space.

üéØ Main Results

  • Limits of Vector-Valued Functions: The limit of a vector-valued function $\mathbf{r}(t)$ as $t$ approaches a value is defined similarly to the limit of real-valued functions, using the norm (magnitude) of the difference between vectors.
  • Continuity: A vector-valued function $\mathbf{r}(t)$ is continuous at a point if the limit of $\mathbf{r}(t)$ as $t$ approaches that point equals the function's value at that point.
  • Derivatives: The derivative of a vector-valued function $\mathbf{r}(t)$ measures the instantaneous rate of change of the function. It is computed component-wise and provides a vector that is tangent to the graph of the function at each point.
  • Tangent Vectors and Lines: The tangent vector to the graph of $\mathbf{r}(t)$ at a point is parallel to the derivative $\mathbf{r}'(t)$. The equation of the tangent line at a point can be found using this tangent vector.
  • Integration: Integrals of vector-valued functions are defined similarly to real-valued functions. The definite integral of $\mathbf{r}(t)$ over an interval provides the displacement, and the arc length of the graph of $\mathbf{r}(t)$ can be found by integrating the magnitude of its derivative.
  • Properties of Derivatives: The properties of derivatives, such as linearity and the product and chain rules, extend to vector-valued functions, allowing us to perform calculus operations component-wise.

üéØ Key Exercise Explanations

üìù Derivative of Vector Function

Problem: Find $\mathbf{r}'(t)$ for $\mathbf{r}(t) = \langle t^2, \sin(t), e^t \rangle$

Key Concept: By Definition 12.2.8, we differentiate each component separately.

Solution Method:

  1. Component-wise differentiation: Differentiate each component separately
  2. First component: $\frac{d}{dt}[t^2] = 2t$
  3. Second component: $\frac{d}{dt}[\sin(t)] = \cos(t)$
  4. Third component: $\frac{d}{dt}[e^t] = e^t$

Answer: $\mathbf{r}'(t) = \langle 2t, \cos(t), e^t \rangle$

Why this works: Theorem 12.2.9 guarantees that a vector function is differentiable if each component is differentiable. For more complex operations, see Theorem 12.2.20 on derivative properties.

üìä Tangent Line Equation

Problem: Find the tangent line to $\mathbf{r}(t) = \langle t^2, t^3 \rangle$ at $t = 1$

Solution Method:

  1. Find point: $\mathbf{r}(1) = \langle 1, 1 \rangle$
  2. Find derivative: $\mathbf{r}'(t) = \langle 2t, 3t^2 \rangle$
  3. Evaluate derivative: $\mathbf{r}'(1) = \langle 2, 3 \rangle$
  4. Write parametric equation: $\mathbf{L}(s) = \mathbf{r}(1) + s \cdot \mathbf{r}'(1)$

Answer: $\mathbf{L}(s) = \langle 1, 1 \rangle + s\langle 2, 3 \rangle = \langle 1 + 2s, 1 + 3s \rangle$

üßÆ Integration of Vector Functions

Problem: Evaluate $\int_0^1 \langle 2t, t^2, 1 \rangle \, dt$

Solution Method:

  1. Component-wise integration: Integrate each component separately
  2. First component: $\int_0^1 2t \, dt = [t^2]_0^1 = 1$
  3. Second component: $\int_0^1 t^2 \, dt = [\frac{t^3}{3}]_0^1 = \frac{1}{3}$
  4. Third component: $\int_0^1 1 \, dt = [t]_0^1 = 1$

Answer: $\langle 1, \frac{1}{3}, 1 \rangle$
