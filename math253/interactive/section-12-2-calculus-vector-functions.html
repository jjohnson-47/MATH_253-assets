üìö Summary

In this section, we delve into the calculus of vector-valued functions. We extend the concepts of limits, derivatives, and integrals from real-valued functions to vector-valued functions, enabling us to analyze these functions with the tools of calculus. This section lays the groundwork for understanding how calculus applies to vector functions and sets the stage for exploring motion in space.

üéØ Main Results

  • Limits of Vector-Valued Functions: The limit of a vector-valued function $\mathbf{r}(t)$ as $t$ approaches a value is defined similarly to the limit of real-valued functions, using the norm (magnitude) of the difference between vectors.
  • Continuity: A vector-valued function $\mathbf{r}(t)$ is continuous at a point if the limit of $\mathbf{r}(t)$ as $t$ approaches that point equals the function's value at that point.
  • Derivatives: The derivative of a vector-valued function $\mathbf{r}(t)$ measures the instantaneous rate of change of the function. It is computed component-wise and provides a vector that is tangent to the graph of the function at each point.
  • Tangent Vectors and Lines: The tangent vector to the graph of $\mathbf{r}(t)$ at a point is parallel to the derivative $\mathbf{r}'(t)$. The equation of the tangent line at a point can be found using this tangent vector.
  • Integration: Integrals of vector-valued functions are defined similarly to real-valued functions. The definite integral of $\mathbf{r}(t)$ over an interval provides the displacement, and the arc length of the graph of $\mathbf{r}(t)$ can be found by integrating the magnitude of its derivative.
  • Properties of Derivatives: The properties of derivatives, such as linearity and the product and chain rules, extend to vector-valued functions, allowing us to perform calculus operations component-wise.

üéØ Key Exercise Explanations

üìù Derivative of a Vector-Valued Function: Worked Example üåü

The Problem: Given the vector-valued function $\mathbf{r}(t) = \langle t^2, \sin(t), e^t \rangle$, find its derivative $\mathbf{r}'(t)$.

üß† Conceptual Foundation: How do we differentiate vector functions?

To find the derivative of a vector-valued function, we rely on Definition 12.2.8. This definition states that if $\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle$, where each component function is differentiable, then:

$\mathbf{r}'(t) = \langle f'(t), g'(t), h'(t) \rangle$

This component-wise differentiation is reinforced by Theorem 12.2.9, which states that a vector function is differentiable if and only if each component is differentiable.

üìù Solution Steps:

1. Identify component functions: For $\mathbf{r}(t) = \langle t^2, \sin(t), e^t \rangle$, we have $f(t) = t^2$, $g(t) = \sin(t)$, and $h(t) = e^t$.

2. Differentiate each component:

  • First component: $f'(t) = \frac{d}{dt}(t^2) = 2t$ (power rule)
  • Second component: $g'(t) = \frac{d}{dt}(\sin(t)) = \cos(t)$ (trigonometric derivative)
  • Third component: $h'(t) = \frac{d}{dt}(e^t) = e^t$ (exponential derivative)

3. Construct the derivative vector: Combining these results gives us:

$\mathbf{r}'(t) = \langle 2t, \cos(t), e^t \rangle$

üí° Further Context:

While this example uses straightforward component-wise differentiation, Theorem 12.2.20 provides important properties for more complex operations, such as derivatives of sums, scalar multiples, and products of vector functions. These properties ensure differentiation works predictably with vector operations.

üìä Tangent Line to a Vector Function: Geometric Insight üåü

The Problem: Find the equation of the tangent line to the vector-valued function $\mathbf{r}(t) = \langle t^2, t^3 \rangle$ at the point where $t = 1$.

üß† Conceptual Foundation: What is a tangent line to a vector function?

When we have a vector-valued function $\mathbf{r}(t)$, its graph traces out a curve in space. At any point on this curve, we can find a tangent line that "best approximates" the curve at that specific location. The direction of this tangent line is given by the derivative vector $\mathbf{r}'(t)$, which points in the direction of instantaneous motion along the curve.

The parametric equation of a tangent line follows the general form: $\mathbf{L}(s) = \mathbf{r}(t_0) + s \cdot \mathbf{r}'(t_0)$, where $\mathbf{r}(t_0)$ is the point of tangency and $\mathbf{r}'(t_0)$ is the direction vector.

üìù Solution Process:

Step 1 - Find the point of tangency: We need to evaluate $\mathbf{r}(t)$ at $t = 1$ to find where the tangent line touches the curve. Substituting $t = 1$ into $\mathbf{r}(t) = \langle t^2, t^3 \rangle$ gives us $\mathbf{r}(1) = \langle 1^2, 1^3 \rangle = \langle 1, 1 \rangle$. This point $(1, 1)$ is where our tangent line will touch the curve.

Step 2 - Find the direction vector: The direction of the tangent line is determined by the derivative $\mathbf{r}'(t)$. Using component-wise differentiation: $\mathbf{r}'(t) = \langle \frac{d}{dt}(t^2), \frac{d}{dt}(t^3) \rangle = \langle 2t, 3t^2 \rangle$. At $t = 1$, this becomes $\mathbf{r}'(1) = \langle 2(1), 3(1)^2 \rangle = \langle 2, 3 \rangle$.

Step 3 - Construct the tangent line equation: Using the standard form $\mathbf{L}(s) = \mathbf{r}(t_0) + s \cdot \mathbf{r}'(t_0)$, we substitute our values:

$\mathbf{L}(s) = \langle 1, 1 \rangle + s \langle 2, 3 \rangle = \langle 1 + 2s, 1 + 3s \rangle$

üí° Geometric Interpretation:

The tangent line $\mathbf{L}(s) = \langle 1 + 2s, 1 + 3s \rangle$ starts at the point $(1, 1)$ when $s = 0$, and moves in the direction of the vector $\langle 2, 3 \rangle$. This means that for every 2 units we move in the x-direction, we move 3 units in the y-direction, creating a line with slope $\frac{3}{2}$. This tangent line provides the best linear approximation to our original curve $\mathbf{r}(t) = \langle t^2, t^3 \rangle$ near the point $t = 1$.

üßÆ Integration of Vector Functions: Finding Displacement üåü

The Problem: Evaluate the definite integral $\int_0^1 \langle 2t, t^2, 1 \rangle \, dt$ and interpret its meaning.

üß† Conceptual Foundation: What does integrating a vector function mean?

When we integrate a vector-valued function, we're essentially finding the "accumulation" of the vector field over the given interval. If $\mathbf{v}(t)$ represents a velocity vector, then $\int_a^b \mathbf{v}(t) \, dt$ gives us the displacement (net change in position) from time $t = a$ to $t = b$. This is the vector analog of how integrating a scalar function gives us the area under the curve.

The key insight is that integration of vector functions is performed component-wise, just like differentiation. Each component is integrated independently using the familiar techniques from single-variable calculus.

üìù Solution Process:

Component-wise integration approach: For the vector function $\langle 2t, t^2, 1 \rangle$, we integrate each component separately over the interval $[0, 1]$.

First component: $\int_0^1 2t \, dt$. Using the power rule, we find the antiderivative: $\int 2t \, dt = t^2 + C$. Applying the fundamental theorem of calculus: $[t^2]_0^1 = 1^2 - 0^2 = 1$.

Second component: $\int_0^1 t^2 \, dt$. The antiderivative of $t^2$ is $\frac{t^3}{3}$, so we evaluate: $[\frac{t^3}{3}]_0^1 = \frac{1^3}{3} - \frac{0^3}{3} = \frac{1}{3}$.

Third component: $\int_0^1 1 \, dt$. The antiderivative of the constant function 1 is simply $t$, giving us: $[t]_0^1 = 1 - 0 = 1$.

Combining the results: The final answer is the vector formed by these three integrated components:

$\int_0^1 \langle 2t, t^2, 1 \rangle \, dt = \langle 1, \frac{1}{3}, 1 \rangle$

üí° Physical Interpretation:

If we interpret $\langle 2t, t^2, 1 \rangle$ as a velocity vector (with units of distance per time), then our result $\langle 1, \frac{1}{3}, 1 \rangle$ represents the displacement vector over the time interval from $t = 0$ to $t = 1$. This means an object following this velocity pattern would move 1 unit in the x-direction, $\frac{1}{3}$ unit in the y-direction, and 1 unit in the z-direction. The integration process has converted instantaneous rates of change into total accumulated change.
