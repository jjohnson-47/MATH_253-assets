üìö Summary

In this section, we explore how vector-valued functions are used to describe the motion of objects in the plane or in space. We examine how calculus concepts such as derivatives and integrals help us understand velocity, acceleration, and displacement, providing a mathematical framework for analyzing motion.

üéØ Main Results

  • Velocity, Speed, and Acceleration:
    • Velocity: The derivative of the position function $\mathbf{r}(t)$, denoted $\mathbf{v}(t) = \mathbf{r}'(t)$, represents the instantaneous rate of change of position.
    • Speed: The magnitude of the velocity vector, given by $\|\mathbf{v}(t)\|$.
    • Acceleration: The derivative of the velocity function, denoted $\mathbf{a}(t) = \mathbf{v}'(t) = \mathbf{r}''(t)$.
  • Projectile Motion: The position function of a projectile, given initial position $\mathbf{r}_0$, initial speed $v_0$, and angle of elevation $\theta$, is:
    $$\mathbf{r}(t) = \mathbf{r}_0 + v_0 t \mathbf{u} - \frac{1}{2} g t^2 \mathbf{j}$$
    where $\mathbf{u} = \cos(\theta) \mathbf{i} + \sin(\theta) \mathbf{j}$ and $g$ is the gravitational constant.
  • Distance Traveled: The total distance traveled by an object with velocity function $\mathbf{v}(t)$ over the interval $[a, b]$ is:
    $$\text{Distance traveled} = \int_a^b \|\mathbf{v}(t)\| \, dt$$
  • Average Speed and Velocity:
    • Average Speed: The distance traveled divided by the total travel time:
      $$\text{Average speed} = \frac{\int_a^b \|\mathbf{v}(t)\| \, dt}{b - a}$$
    • Average Velocity: The displacement divided by the total travel time:
      $$\text{Average velocity} = \frac{\mathbf{r}(b) - \mathbf{r}(a)}{b - a}$$

üéØ Key Exercise Explanations

üöÄ Projectile Motion Analysis

Problem: A projectile is launched from ground level with initial speed 100 ft/s at an angle of 45¬∞. Find the maximum height and range.

Solution Method:

  1. Set up position function: $\mathbf{r}(t) = \langle 100\cos(45¬∞)t, 100\sin(45¬∞)t - 16t^2 \rangle$
  2. Simplify components: $\mathbf{r}(t) = \langle 50\sqrt{2}t, 50\sqrt{2}t - 16t^2 \rangle$
  3. Find maximum height: Set $y'(t) = 50\sqrt{2} - 32t = 0$, so $t = \frac{50\sqrt{2}}{32}$
  4. Calculate max height: $y_{\text{max}} = 50\sqrt{2} \cdot \frac{50\sqrt{2}}{32} - 16 \left(\frac{50\sqrt{2}}{32}\right)^2 = 78.125$ ft
  5. Find range: Set $y(t) = 0$, solve for $t$, then find $x(t)$ at that time

Answer: Maximum height = 78.125 ft, Range = 312.5 ft

üìä Velocity and Acceleration

Problem: Given $\mathbf{r}(t) = \langle t^2, t^3, t \rangle$, find velocity, speed, and acceleration at $t = 2$

Solution Method:

  1. Find velocity: $\mathbf{v}(t) = \mathbf{r}'(t) = \langle 2t, 3t^2, 1 \rangle$
  2. Evaluate at t=2: $\mathbf{v}(2) = \langle 4, 12, 1 \rangle$
  3. Find speed: $\|\mathbf{v}(2)\| = \sqrt{4^2 + 12^2 + 1^2} = \sqrt{161}$
  4. Find acceleration: $\mathbf{a}(t) = \mathbf{v}'(t) = \langle 2, 6t, 0 \rangle$
  5. Evaluate at t=2: $\mathbf{a}(2) = \langle 2, 12, 0 \rangle$

Answer: $\mathbf{v}(2) = \langle 4, 12, 1 \rangle$, Speed = $\sqrt{161}$, $\mathbf{a}(2) = \langle 2, 12, 0 \rangle$

üìè Arc Length of 3D Curves: Understanding Distance Along Paths üåü

The Problem: Find the arc length of the 3D helical curve $\mathbf{r}(t) = \langle 3\cos(t), 3\sin(t), 4t \rangle$ from $t = 0$ to $t = 2\pi$.

üß† Conceptual Foundation: What is arc length for space curves?

Arc length measures the actual distance traveled along a curve, not the straight-line displacement from start to finish. For a vector-valued function $\mathbf{r}(t)$, we can think of arc length as the distance a particle would travel if it followed the path traced by $\mathbf{r}(t)$.

The fundamental principle is that we integrate the speed (magnitude of velocity) over time: $\text{Arc Length} = \int_a^b \|\mathbf{r}'(t)\| \, dt$. This is the 3D generalization of the arc length formula from single-variable calculus.

üìù Solution Process:

Step 1 - Understand the curve: Our function $\mathbf{r}(t) = \langle 3\cos(t), 3\sin(t), 4t \rangle$ represents a helix. The first two components $\langle 3\cos(t), 3\sin(t) \rangle$ trace a circle of radius 3 in the xy-plane, while the third component $4t$ creates vertical motion, forming a spiral staircase-like path.

Step 2 - Find the velocity vector: Taking the derivative component-wise: $\mathbf{r}'(t) = \langle \frac{d}{dt}(3\cos(t)), \frac{d}{dt}(3\sin(t)), \frac{d}{dt}(4t) \rangle = \langle -3\sin(t), 3\cos(t), 4 \rangle$.

Step 3 - Calculate the speed: The speed is the magnitude of the velocity vector: $$\|\mathbf{r}'(t)\| = \sqrt{(-3\sin(t))^2 + (3\cos(t))^2 + 4^2} = \sqrt{9\sin^2(t) + 9\cos^2(t) + 16}$$ Using the identity $\sin^2(t) + \cos^2(t) = 1$, this simplifies to $\sqrt{9(1) + 16} = \sqrt{25} = 5$.

Step 4 - Set up and evaluate the integral: Since the speed is constant, the arc length integral becomes:

$\text{Arc Length} = \int_0^{2\pi} 5 \, dt = 5[t]_0^{2\pi} = 5(2\pi - 0) = 10\pi$

üí° Geometric Interpretation:

The result $10\pi \approx 31.42$ units means that if you were to "unwrap" this helical path and lay it straight, it would measure exactly $10\pi$ units long. This makes intuitive sense: in one complete revolution ($t$ from $0$ to $2\pi$), the particle moves around a circle of radius 3 (circumference $6\pi$) while simultaneously rising $4(2\pi) = 8\pi$ units vertically. The actual path length accounts for both the circular and vertical motion components.
